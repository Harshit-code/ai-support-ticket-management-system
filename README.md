# AI Support Ticket Management System

A full-stack support ticket management system.

## Tech Stack

| Layer    | Technology                          |
|----------|-------------------------------------|
| Backend  | ASP.NET Core 8 Web API              |
| ORM      | Entity Framework Core 8             |
| Frontend | React 18 + TypeScript + Vite        |
